# BI Salary Insights Analysis

This project focuses on applying Business Intelligence (BI) techniques to analyze workforce salary data and extract meaningful insights.

## Project Objective
The objective of this project is to analyze employee salary data in order to identify key factors influencing compensation. The analysis supports data-driven decision making related to salary benchmarking, workforce planning, and employee fairness.

## Dataset
- Ask a Manager Salary Survey (Cleaned and Raw Versions)
- Dataset includes information related to roles, experience, industry, and salary ranges.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
- Power BI

## Project Steps
- Data Cleaning and Preparation
- Exploratory Data Analysis (EDA)
- Salary Distribution and Trend Analysis
- Insight Generation and Visualization

## Key Outcomes
- Identified patterns and trends in salary distribution
- Analyzed the impact of different factors on employee compensation
- Generated insights to support business and HR decision-making
